# SmartFix - Complaint Management System

SmartFix is a modern complaint and service management platform designed to streamline communication between users, managers, and technicians.

## Features
- **User Dashboard**: Submit and track repair requests.
- **Manager Dashboard**: Analyze AI-driven insights and assign technicians.
- **Technician Portal**: Manage assigned tasks and update status.
- **Real-time Updates**: Socket.io integration for instant status changes.
- **AI Analysis**: Automated issue classification and requirement prediction.

## Tech Stack
- **Frontend**: React (Vite), Redux Toolkit, CSS3
- **Backend**: Node.js, Express, MongoDB, Socket.io
- **AI**: Gemini Pro API
